Väkommna!

Välkomna till vårt gymnasiearbete om cybersäkerhet och etisk hacking. Vi är två elever som går teknik och valde dator- och nätverksteknik, samt programmering som vårt ämne för gymnasiearbete. På följande sida kommer det finnas olika tutorials om följande ämnen; fiskning, wi-fi, anonymitet, skydd och extra såsom flipper zero. Ämnen valdes utifrån vårt formulär som du kan titta på i samma sida.

Dina första steg | Virtuell Maskin

En Virtual Machine (VM) är som en dator inuti din dator, den låter dig köra olika operativsystem på samma maskin. Kali Linux är ett specialiserat datorkompletterande program som används av säkerhetsexperter för att testa och förbättra datorsystemssäkerhet. I denna videon kommer du få se hur du installerar och sätter upp dessa program på både Windows och Mac. Nedan kan du se systemkrav som din dator måste uppfylla: macOS 10.15 eller senare En 64-bitars Intel-baserad Mac eller En 64-bitars Windows-dator Minst 4 GB RAM Rekommenderas minst 20-30 GB ledigt lagringsutrymme för installationen av både Virtual Machine och Kali Linux, kan behövas mer med möjliga tillägg.
